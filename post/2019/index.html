# 2019

2020-01-25

网上看了些年终总结，大多数还是打广告性质的，讲述下自己这一年干了啥，作为论述下自己很牛逼的证据，确实，一眼看下来除了牛逼，嘴巴还要惊讶的被大张开来，暗暗和自己对比了下，咋人与人之间差距这么大呢。

我在以前看到自己写的旧笔记，日记时都会想删掉，心里还要疑问下，这个傻逼是我？慢慢的感觉到时间飞逝，留下来一些记录也挺有意思的，菜的自己也是自己呢，所以有了这篇总结呢。

想来从 2018 年 6 月毕业至今马上就要工作两年了，和每次回想以前的事一样，都会感叹时间过的之快。

感觉现在技术水平算是入了个门吧，从刚开始来公司的时候，idea 用不来，上网查教程到现在上网查看面试题，学习的过程就是这个样子吧。

第一次看到组长代码里用 BeanUtils 的 copyProperties 方法，第一次看到同事用 ApplicationContextAware 时候的惊讶，看到同事封装 react 写个类的时候，看到公司内部使用的框架是主管写的时候，我小小的内心都被震撼了。有一次听到隔壁组的同事说分布式锁的时候，真的是浑身抖擞了一下，太牛逼了，心里不断的发出感叹。

然后就是稳步进行中的搬砖事业，被 react 前端吊打的过程，从 java 基础开始恶补知识。

印象更深的就是今年年中的时候组里来了个实习生，想下同期的自己在干啥，更加感觉自己还是太菜了，还要加把劲，总又感觉有点奇怪，算来也 25 岁了，咋还要每天学习。。

年中以后换了个组，开始写点超出我水平的代码，写都写不来，只能去抄代码。。从看 nacos 代码时候被代码里的心跳代码惊艳了，顿时感觉之前不知道在干啥，憨憨的一样。好在虽然超时了，组长也只能背后低估下超时了超时了，也不会说啥。。

慢慢心里感觉公司 low 的感觉越来越强烈，code review 都没有啊，还要老是强调代码规范，同事maven 打个 jar 包老是翻车坑我，公司的文化也越来越熟悉，熟悉后有点失望。

这一年绩效也不好看，心态时常爆炸。好在现在能稳的住了。

我觉得我还是幸运的，在公司的时候主要还是在学习，有时干些超出自己水平的事。更难可贵的是心态得到了锻炼，这种财富还是更加珍贵的。
